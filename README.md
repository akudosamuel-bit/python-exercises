
# Python Data Structures Exercises — Akudo Samuel

This repository contains implementations of the Python data-structure exercises from the submitted assignment. Each module groups related functions (numeric types, strings, lists, dictionaries, tuples & sets, and advanced tasks). Run individual files to see simple demos.

## Structure

- `exercises/01_numeric_types.py` — numeric functions (area, conversions, arithmetic helpers)
- `exercises/02_strings.py` — string operations (reverse, case change, extract, etc.)
- `exercises/03_lists.py` — list operations (append, insert, remove, slicing, etc.)
- `exercises/04_dictionaries.py` — dictionary operations (create, update, merge, safe get)
- `exercises/05_tuples_sets.py` — tuples and sets utilities
- `exercises/06_advanced.py` — nested data and course-related helpers
- `tests/` — example unit tests

## Usage

Run a module directly:

```bash
python exercises/02_strings.py
